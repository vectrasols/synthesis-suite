# Security Policy

## Supported Versions

Security fixes are intended for the latest released version.

## Reporting a Vulnerability

If you find a vulnerability, please report it privately to the project maintainer. Include:

- Affected version or commit
- Steps to reproduce
- Impact and any known workaround

Do not publish exploit details until a fix or mitigation is available.

## Local Data

Synthesis is designed for local desktop analysis. Uploaded/imported datasets are processed by a local FastAPI backend spawned by Electron. Review any URL-import source before loading remote data.
